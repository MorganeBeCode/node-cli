# Node.JS: Holidates CLI

- Author: Morgane Jansen
- Version: 1.1.0

### Description

A command line tool that takes a country name as parameter then return the national holidays for the current yeat.

### Installation

> \$ npm install @morganebecode/holidates

### How to use

> \$ holidates [country]
